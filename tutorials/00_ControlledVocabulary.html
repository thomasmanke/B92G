Query ECL Template

Snomed CT codes form a large ontology. Queries into this data and be formulated with a so called Expression Constraint Language (ECL).

Code 
def query_ecl(ecl_expression):
    """
    Sends ECL query to FHIR endpoint
    Returns list of concepts names     
    """
    params = {
        "url": f"http://snomed.info/sct?fhir_vs=ecl/{ecl_expression}",
        "count": 500
    }
    r = requests.get(FHIR_URL, params=params)
    r.raise_for_status()
    
    data = r.json() 
    # data.keys(): resourceType, url (=query), expansion

    # return list of concepts data["expansion"]["contains"]
    return data.get("expansion", {}).get("contains", [])

#### Query ECL Template

Snomed CT codes form a large ontology. Queries into this data and be formulated with a so called Expression Constraint Language (ECL).

```{python}
#| label: fhir_with_ecl

def query_ecl(ecl_expression):
    """
    Sends ECL query to FHIR endpoint
    Returns list of concepts names     
    """
    params = {
        "url": f"http://snomed.info/sct?fhir_vs=ecl/{ecl_expression}",
        "count": 500
    }
    r = requests.get(FHIR_URL, params=params)
    r.raise_for_status()
    
    data = r.json() 
    # data.keys(): resourceType, url (=query), expansion

    # return list of concepts data["expansion"]["contains"]
    return data.get("expansion", {}).get("contains", [])
```

Simple Usage Example: retrieve a single concept

```{python}
#| label: ECL_simple_use
disease = "73211009" # Diabetes mellitus

query = f"{disease}" # very basic ECL query
results = query_ecl(query)

pd.DataFrame(results)
```


#### Complex ECL queries

More complicated ECL queries can also be used to search and retrieve multiple (related) concepts and attributes from the Snomed Ontology, e.g.

- concept: "{code}"
- parents: ">{code}"
- children: "<{code}"
- more examples: [ECL quick reference](https://docs.snomed.org/snomed-ct-specifications/snomed-ct-expression-constraint-language/appendices/appendix-d-ecl-quick-reference){target="_blank" rel="noopener"}

### Ontology Task
- Find the Snomed CT code for disease "Obesity"
- Find the ECL expression to determine
    - all parents concept for disease "obesity"
    - all children diseases for disease "obesity"
- Determine all attributes ("Due to") for disease "Diabetes Mellitus"

    - [ECL help](https://docs.snomed.org/snomed-ct-specifications/snomed-ct-expression-constraint-language/appendices/appendix-d-ecl-quick-reference){target="_blank" rel="noopener"}

```{python}
#| label: ecf_task
#| eval: false
disease="..."                # find code for "obesity"

# find ECL expression for 
parents  = query_ecl(f"...") # children (=direct descendants)
children = query_ecl(f"...") # parents (=direct ancestors)


print(f'Retrieved: {disease}')
print('Parents:  \n', pd.DataFrame(parents))
print('Children: \n', pd.DataFrame(children))

disease = "..."              # find code for Diabetes mellitus
attrib  = "..."              # find code for attribute "due to"

```

```{python}
#| label: ecf_solution
#| eval: false
#| echo: false
disease="414916001" # Obesity
ecl=f"<!{disease}" # get chilrdren of disease
parents  = query_ecl(f">!{disease}")
children = query_ecl(f"<!{disease}")

print(f'Retrieved: {disease}')
print('Parents:  \n', pd.DataFrame(parents))
print('Children: \n', pd.DataFrame(children))

disease = "73211009"     # diabetes mellitus
attrib  = "42752001"     # "due to"
results = query_ecl(f"<<{disease}.{attrib}")  
print('Attributes: \n', pd.DataFrame(results))
```

#| label: ecl_more_complex_examples

query_ecl(f"(>>{d1}) AND (>>{d2})") # all common ancestors
